# Deals & Steals Website V1

A Next.js + Tailwind homepage for Deals & Steals.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload these files to the repo.
3. In Vercel, click **Add New Project** and import the repo.
4. Deploy.
5. Add your domain: `dealsandstealsmd.com`.
